Task6 Web 界面 - 运行与测试说明
==============================================
说明: 当前无针对 Dash 的自动化 pytest；采用手动冒烟验证。

运行命令
----------------------------------------------
在项目根目录执行：

  python -m src.web.app

预期输出
----------------------------------------------
终端应出现类似：

  Dash is running on http://0.0.0.0:8050/

  * Serving Flask app 'app'
  * Debug mode: on

访问地址
----------------------------------------------
浏览器打开： http://127.0.0.1:8050/  或  http://localhost:8050/
（请勿使用 http://0.0.0.0:8050/ ，否则可能 ERR_ADDRESS_INVALID）

手动冒烟检查项
----------------------------------------------
1. 页面加载：侧栏显示「会话」与「新建会话」，主区有输入框与「发送」。
2. 首次加载：默认一个「新会话」，消息区为空或提示「暂无消息」。
3. 发送消息：输入问题后点「发送」，出现 loading，随后显示用户消息与助手回复（含 Markdown、Skill 标签）。
4. 推荐练习：若回复含「## 推荐练习」，该段以卡片形式展示。
5. 新建会话：点「新建会话」，侧栏多一项，主区消息清空。
6. 切换会话：点侧栏另一会话，主区切换为该校话的消息列表。
7. 导出：点「导出当前会话」，浏览器下载 JSON 文件。
